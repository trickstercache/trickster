# Security

If you've found a vulnerability or a potential vulnerability in Trickster, please let us know using the 'Report a vulnerability' button under the [Security and quality > Advisories](https://github.com/trickstercache/trickster/security/advisories) section of the project. Per the [Contributing Guidelines](./CONTRIBUTING.md), vulnerabilities must be reported by human contributors from human-controlled GitHub accounts. We do not accept vulnerability reports from automated/bot accounts, regardless of their technical merit.
